# UI 样式规范（襄行摩友小程序 + Wot-Starter）
## 一、色彩规范（暗黑模式强制）
- 主色：#2ED573（骑行绿，按钮/选中态）
- 辅助色：#FF7A00（荧光橙，警示）
- 背景：主背景 #121212、辅助背景 #1E1E1E、悬浮背景 #272727
- 文字：主文字 #FFFFFF、次文字 #CCCCCC、弱提示 #8D99AE
- 边框/分割线：#333333

## 二、组件规范
1. 100% 使用 Woto UI 原生组件（`wd-*` 前缀），禁止自定义基础组件；
2. 暗黑模式全程生效，禁止浅色背景/高亮元素；
3. UnoCSS 仅用于布局/间距/圆角，禁止覆盖 Woto UI 组件样式；
4. 样式强制开启 `scoped`，禁止全局样式污染。

## 三、尺寸规范（rpx 统一）
- 图标：36rpx（常规）、40rpx（导航）、60rpx（首页大图标）
- 文字：20rpx（辅助）、22rpx（常规）、24rpx（标题）、28rpx（主标题）
- 圆角：8rpx（按钮）、12rpx（卡片）、16rpx（大卡片）
- 边距：16rpx（基础）、20rpx（常规）、32rpx（页面边距）

## 四、交互规范
1. 点击反馈：背景变 #272727 + 轻微缩放 1.02 倍，过渡动画 100ms；
2. 加载态：统一使用 Woto UI `wd-loading`，禁止自定义加载动画；
3. 异常提示：统一使用 Woto UI `wd-toast`，文案简洁；
4. 适配微信小程序安全区域，避免刘海/小黑条遮挡。
