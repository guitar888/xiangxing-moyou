# 襄行摩友小程序 - 开发规范（需求版）

## 一、核心定位与合规（仅摘要）
- 纯公益工具，无交易无社交无导流
- 活动仅展示回顾，通过微信群运营
- 所有页面必须加合规声明

## 二、功能明细（仅业务描述）
### 5个Tab页面
- 首页：定位、快捷入口、公告、数据概览、活动、合规声明
- 骑行地图：腾讯地图、路线/打卡点、定位导航筛选
- 骑行数据：图表、统计、记录、海报生成
- 活动工具：列表筛选、AA计算器、路线跳转、群公告引导
- 我的：数据、设置、暗黑、清除缓存、无联系方式

### 拓展分包
- 经典线路、骑行安全、工具合集、天气查询、禁摩限行、物品互助

## 三、UI规范（仅摘要）
- 暗黑默认、极简、襄阳元素
- 主色#2ED573、橙#FF7A00、灰#8D99AE
- 字体、字号、字重统一标准

## 四、审核与部署
- 类目：工具-出行/旅游
- 禁止联系方式、禁止社交交易、内容必须审核
